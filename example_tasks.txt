# Cron 任务示例配置文件
# 格式1: 分 时 日 月 周 任务名称
# 格式2: 任务名称=表达式

# 数据同步任务 - 每15分钟执行一次
*/15 * * * * 数据同步

# 数据库备份 - 每天凌晨2点
0 2 * * * 数据库备份

# 日志清理 - 每天凌晨3点半
30 3 * * * 日志清理

# 工作日晨会 - 周一到周五早上8点
0 8 * * 1-5 工作日晨会

# 周末巡检 - 周六周日早上9点
0 9 * * 0,6 周末巡检

# 每小时整点报表
0 * * * * 整点报表

# 每30分钟健康检查
0,30 * * * * 健康检查

# 月末结账 - 每月1号凌晨1点
0 1 1 * * 月末结账

# 每周三下午2点半的评审会议
30 14 * * 3 周度评审
